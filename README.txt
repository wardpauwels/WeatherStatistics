Om dit project te kunnen runnen, gebruik het commando "node app.js" in de root van deze folder.

Dit project bestaat uit een website en een zelfgeschreven api, allebij met een node back-end.

De api wordt aangesproken door de website om zijn gegevens op te halen. De api wordt ook gebruikt door het project van "Internet of Things" om gegevens naartoe te sturen.

De gegevens die de applicatie gebruikt, zijn dus real-time.


De database wordt gehost op http://mongodb.com/ en de applicatie zelf op http://ibm.com/

Link naar applicatie: http://weatherstatistics.eu-gb.mybluemix.net/
